## Emacs 관련 정보 정리 :)

### emacs 기본 설정 파일 경로 

`~/.emacs`, `~/emacs.d/init.el`

### emacs daemon

http://www.emacswiki.org/emacs/EmacsAsDaemon


```
emacs --daemon
```


### 단축키 정리 

#### 버퍼 열기

```
C-x
```

#### 버퍼 닫기

```
C-x k
```

#### 창간 이동 

```
C-x o
```

#### 윈도우 닫기 
```
C-x 0 (선택된 윈도우 닫기)
```

```
C-x 1 (모든 창 닫기)
```

#### 검색

```
C-s
```
